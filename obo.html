Manual de Usuario - Compilador OBO

Bienvenido al manual de usuario del compilador OBO. Aquí encontrarás toda la información necesaria para utilizar el compilador del lenguaje OBO de manera efectiva.

¿Qué es OBO?

OBO es un compilador diseñado para procesar el lenguaje de programación OBO. Este lenguaje está orientado a simplicidad y eficiencia, ideal para aplicaciones ligeras y aprendizaje básico de programación.

Instalación

Para instalar el compilador OBO, sigue estos pasos:

  1. Descarga el archivo ejecutable desde el sitio oficial: www.ejemplo.com.
  2. Ejecuta el instalador y sigue las instrucciones en pantalla.
  3. Asegúrate de que la ruta del compilador esté incluida en las variables de entorno del sistema.

Cómo usar el compilador OBO

El compilador OBO utiliza la línea de comandos para compilar programas escritos en lenguaje OBO. Aquí tienes un ejemplo básico de uso:

1. Crear un archivo OBO

Escribe tu código en un archivo con extensión .obo. Por ejemplo:

print "¡Hola, mundo!"

2. Compilar el archivo

Abre la terminal y ejecuta el siguiente comando:

obo myprogram.obo

Esto generará un archivo ejecutable en el mismo directorio.

3. Ejecutar el programa

Ejecuta el archivo generado con el comando correspondiente a tu sistema operativo:

./myprogram

Características del lenguaje OBO

  • Sintaxis minimalista.
  • Soporte para estructuras básicas de control como bucles y condicionales.
  • Operaciones matemáticas simples.
  • Gestión básica de cadenas y archivos.

Comandos y opciones del compilador

El compilador OBO soporta las siguientes opciones:

Comando Descripción
-v o --version Muestra la versión del compilador.
-h o --help Muestra la ayuda del compilador.
-o [archivo] Especifica el nombre del archivo de salida.

Resolución de problemas comunes

Si encuentras problemas al usar el compilador OBO, consulta esta sección:

  • Error: "Comando no encontrado"
    Asegúrate de que la ruta del compilador está incluida en las variables de entorno.
  • Error: "Archivo no encontrado"
    Verifica que el archivo .obo exista y esté en el directorio correcto.
  • Salida inesperada
    Revisa la sintaxis de tu programa para asegurarte de que cumple con las reglas del lenguaje OBO.
